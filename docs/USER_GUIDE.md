# 运行管理系统使用说明（按目标操作步骤）

> 以“想达成什么目的 → 怎么做”方式说明，每个页面给出具体点击与填写步骤。

## 登录 / 退出 / 修改密码
- 登录：浏览器访问系统地址（默认 http://localhost）→ 输入用户名/密码（区分大小写）→ “登录”。
- 修改密码：右上头像/用户名 → “修改密码” → 填原密码/新密码/确认新密码 → “保存”。
- 退出：右上头像/用户名 → “退出登录”。

## 排班（/schedule）
目标：给人员排班或查看排班。
1) 进入“排班表”，选择日期范围或人员筛选后点击“查询”查看现有排班。
2) 新增排班：点击“新增”或“批量新增”（若有），选择人员、日期、班次 → “保存”。
3) 调整/删除：在表格中选中记录，点击“编辑”修改班次或“删除”移除。
4) 如页面提供导入：点击“下载模板”→ 按模板填人员、日期、班次 → 保存 → “导入”→ 选文件 → 确认。

## 用户管理（/user-management）
目标：新增/导入/调整用户。
1) 筛选查看：在顶部输入关键词、公司、角色、部门、场站、状态 → “搜索”。
2) 新增单个：点击“新增用户”→ 填姓名、用户名（必填、3-20 位、唯一）、初始密码、基准角色（必选）、公司/部门/场站（可选）、手机号/邮箱（可选）、状态 → “保存”。
3) 批量导入：
   - “下载模板”到本地。
   - 打开模板，按列填写：用户名（必填、唯一、3-20 字符）、姓名（必填）、公司/部门/场站（可选，需与系统名称一致）、角色（必填，使用模板列出的角色名之一）、手机号（11 位，可选）、邮箱（可选）。
   - 保存 Excel → 点击“批量导入”→ 选择文件 → 确认；若提示错误，按行号修正后重导。
4) 编辑：行尾“编辑”→ 调整基准角色或权限树（基准角色勾选自动带出，可直接勾/去勾菜单与模块）→ “保存”。
5) 重置密码：行尾“重置密码”→ 确认（恢复为默认密码，如 123456）。
6) 删除：行尾“删除”→ 确认。

## 角色管理
目标：配置角色与默认权限。
1) 新增：点击“新增”→ 填名称、编码、基准角色 → 在权限树勾选菜单/模块 → “保存”。
2) 编辑：行尾“编辑”→ 调整信息或权限 → “保存”。
3) 删除：行尾“删除”（内置角色不可删）。

## 组织架构
目标：维护公司/部门/场站。
1) 公司：在公司列表点击“新增/编辑/删除”；填写公司名称等 → “保存”。
2) 部门：在部门列表“新增/编辑/删除”；填写部门名称、所属公司 → “保存”。
3) 场站：在场站列表“新增/编辑/删除”；填写场站名称、类型、所属公司/部门 → “保存”。

## 单价管理
目标：维护单价。
1) 查看/筛选单价列表。
2) 新增/编辑：填写类别、单价、说明 → “保存”。
3) 删除：行尾“删除”→ 确认。（仅单价管理员/管理员可写）

## 消息通知 / 帮助反馈
- 消息通知：查看列表，可“全部标记已读”或行尾“删除”。
- 帮助与反馈：填写标题、内容（可上传截图）→ “提交”。

## 数据报表（/reports）
目标：查看保养/故障统计、趋势、导出。
1) 选择标签：“设备保养统计”或“设备故障统计”。
2) 选择日期范围、场站 → “查询”刷新统计卡片、柱状图/饼图。
3) 全年趋势：在“设备全年趋势”选择设备、年份 → “查询”刷新折线图。
4) 导出：点击“导出 Excel”，按当前筛选导出表格。

## 设备保养
### 设备保养（新版，/equipment-maintenance）
目标：查看并提交保养工单。
1) 筛选日期/场站/状态 → “查询”。
2) 点击工单查看详情；有权限时填写工时/结果/备注 → “提交”或“验收”。

### 保养计划（/maintenance-plan）
目标：创建或维护保养计划。
1) 点击“新增计划”→ 填设备、周期（每日/每周/每月/每年）、执行人/场站、开始时间等 → “保存”。
2) 行尾“编辑”修改，或“删除”移除。

### 保养记录（/maintenance-record）
目标：查看历史保养记录。
1) 按日期/状态筛选 → 查看列表；如有导出按钮可导出。

### 保养任务（/maintenance-task）
目标：员工提交个人保养任务。
1) 选择任务行 → 填写实际工时、结果等 → “提交”。

## 故障与维修
### 故障上报（/fault-report）
目标：报修。
1) 填写设备、故障描述、发生时间、上报人、照片（可选）→ “提交”。

### 故障列表（/fault-list 或 /device-faults）
目标：查看故障。
1) 按报修时间/状态/设备筛选 → “查询”。
2) 行尾“查看”查看详情。

### 维修派发单（/maintenance-dispatch）
目标：派单。
1) 选择待派发故障 → 指定维修负责人/配合人、计划时间 → “保存”。

### 维修工作（/repair-work）
目标：维修人员处理工单。
1) 切换“待处理/进行中/已完成”页签查看任务。
2) 点击任务，填写维修内容、耗材/配件、上传照片（可选）→ 更新状态（开始/完成）→ “保存/提交”。

### 维修记录（/repair-records，/repair-record/new，/repair-record/:id）
目标：完整维护维修记录。
1) 新建：填写设备、故障描述、维修人、材料/耗材明细、结果等 → “保存”或“提交验收”。
2) 查看/编辑：在列表行尾“查看/编辑”；按需要更新并保存。
3) 删除：行尾“删除”（有权限时）。

## 安全 / 卫生
### 安全自检（/safety-self-inspection）
目标：提交自检。
1) 选择场站、日期，勾选检查项，填写备注/照片（可选）→ “提交”。
2) 历史记录可按日期筛选查看。

### 安全他检（/safety-other-inspection）
目标：发起他检。
1) 选择被检人、场站、检查项目，填写结果/备注 → “提交”。

### 安全检查项目管理（/safety-check-management，安全员）
目标：维护检查项目。
1) 批量导入：点击“下载模板”→ 填写工作性质、检查项名称、标准等 → 保存 Excel → “导入”→ 选文件 → 确认（错误提示行号，修正后重导）。
2) 单条维护：列表“新增/编辑/删除”检查项；工作性质同理。

### 安全隐患（/safety-rectification）
目标：提报与整改隐患。
1) 新建隐患：填写场站、隐患描述、整改人、截止时间、照片（可选）→ “提交”。
2) 整改人处理：打开隐患 → 填整改措施、完成时间 → “提交”。
3) 安全员复查：在列表选中 → 填复查结论 → “提交”。

### 卫生自检/他检（/hygiene-self-inspection，/hygiene-other-inspection）
目标：提交卫生检查。
1) 选择场站、检查人/被检人，勾选检查项，填写备注/照片（可选）→ “提交”。

## 岗位工作 / 临时工作
### 岗位工作（/position-work）
目标：填写当日岗位任务。
1) “今日任务”列表中选择任务 → 填完成情况/工时 → “保存”。
2) 管理角色可按人员/日期筛选查看工作记录。

### 临时工作管理（/temporary-tasks）
目标：创建并推进临时任务。
1) 新增：点击“新增”→ 填任务名称、描述、类型、执行人、开始/结束时间 → “保存”。
2) 进度：行尾“开始/完成”更新状态；“编辑”调整信息；“删除”移除。

## 设备管理（/equipment）
目标：维护设备。
1) 查看/筛选设备列表。
2) 新增/编辑：填写设备编号、名称、场站等 → “保存”；有权限可“删除”。
3) 若页面提供导入：下载模板→ 按列填设备编号/名称/场站等 → 保存 → “导入”→ 选文件 → 确认。

## 其他
- PLC 数据（/plc-records）：查看 PLC 采集数据。
- 404：访问不存在地址跳转“页面不存在”，点击返回首页即可。

## 遇到问题
- 页面异常：Ctrl+F5 强制刷新；无效则重新登录。
- 无权限或数据缺失：联系管理员调整角色/场站权限。
- 报错：记录时间、操作步骤、报错提示，反馈给运维或管理员。
